# Jogos em C - Adivinhação e Forca

Projeto acadêmico desenvolvido em linguagem C, contendo dois jogos interativos executados via terminal: um jogo de adivinhação de números e um jogo da forca.

## 🎮 Funcionalidades

### Jogo de Adivinhação

* O usuário tenta adivinhar um número gerado aleatoriamente
* O sistema fornece dicas (maior ou menor) para auxiliar na tentativa

### Jogo da Forca

* O usuário tenta descobrir a palavra secreta
* Controle de tentativas
* Exibição do progresso da palavra
* Ao final do jogo, é possível adicionar uma nova palavra ao banco de palavras, aumentando a variedade do jogo

## 🛠️ Tecnologias utilizadas

* Linguagem C
* Execução via terminal/console

## 📚 Conceitos aplicados

* Lógica de programação
* Estruturas condicionais (if/else)
* Estruturas de repetição (for, while)
* Manipulação de variáveis
* Entrada e saída de dados (scanf/printf)
* Uso de funções
* Geração de números aleatórios (rand)
* Manipulação básica de dados para expansão do jogo (inclusão de novas palavras)

## 🎯 Objetivo

Praticar e consolidar conceitos fundamentais da linguagem C por meio do desenvolvimento de jogos interativos, estimulando a lógica, a interação com o usuário e a evolução do sistema com novas funcionalidades.

💻 Como executar
Certifique-se de ter um compilador C/C++ instalado (como o gcc)
Acesse a pasta do projeto pelo terminal
Compile os arquivos:
Jogo da Forca

gcc Jogo_forca.c -o forca
./forca

Jogo de Adivinhação

gcc "jogo de adivinhação.cpp" -o adivinhacao
./adivinhacao

Siga as instruções exibidas no terminal para jogar
